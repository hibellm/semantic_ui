In order to gain access to any Datasource, you must first have a completed Read & understood.


The process is as follows:

  1. Find the LSO code fo rthe data source you want access for Click here for a list
  2. Go to the LSO portal and enter the code
  3. Save the R&U completion PDF
  4. Using the RWD web form, attach the R&U completion, and request access for the data source.
